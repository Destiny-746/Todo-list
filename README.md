# Todo-list

A simple To-Do list web app built with HTML, CSS, JavaScript.

## Features

- Add new task
- ~~Edit task~~
- Mark as complete
- Mark as incomplete
- Move to trash
- Recover from trash
- Delete forever
- Save on localstorage
- Different tabs
- Change page theme (Light, Dark)
